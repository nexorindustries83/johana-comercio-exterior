# Johana - Pagina Comercio Exterior

Sitio estatico en HTML listo para publicar en Render.

## Publicar en Render

1. Sube este proyecto a tu repositorio de GitHub (manual, como prefieres).
2. En Render, entra a `New +` -> `Blueprint`.
3. Conecta tu repo y selecciona la rama principal.
4. Render detectara `render.yaml` y creara el servicio estatico.
5. Haz clic en `Apply` para desplegar.

## Configuracion usada

- Tipo: `web`
- Runtime: `static`
- Build command: `echo "No build step required"`
- Publish directory: `.`
